🖥️Como usar no SICS após exportar

  2. Abra o SICS → Vendas → Pedido de Venda → Novo Lançamento (9999)
  3. Identifique o cliente manualmente no cadastro do SICS
  4. Use o CSV aberto ao lado como referência para digitação
  5. Salve → Lista de Vendas → clique na venda → Gerar Produção → Imprimir Etiquetas
